# 利用方法 	
石神井公園内で撮った写真を投稿し、他ユーザーとのささやかな交流が持てます。

# 目指した課題解決 	
石神井公園は、東京都練馬区にある都立公園。井の頭池、善福寺池と並び、武蔵野三大湧水池として知られており、平日休日問わず、多くの人々が行楽に訪れる人気のスポットです。
都内でも有数の自然豊かな公園となっており、野鳥の撮影をするために非常に多くのカメラマンで賑わっています。そのカメラマンの人たちが撮影した写真を、誰か他の人達に見てもらう場所を提供しようと考えました。
